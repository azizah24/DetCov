# Deteksi Covid

Aplikasi Deteksi Covid untuk melihat gejala yang diinputkan dengan menu :
- Menu Spalsh screen
- Menu Home 
- Menu Apa Itu Covid-19
- Menu Gejala Covid-19
- Menu Deteksi Covid-19
- Menu Statistik
